# Active Directory Home Lab

This repository documents the creation and administration of a home lab environment built to develop practical experience with Windows Server, Active Directory, networking, PowerShell, and systems administration.

## Objectives

* Gain hands-on experience with Active Directory Domain Services
* Practice Windows Server administration
* Learn Group Policy management
* Configure DNS and networking services
* Implement file sharing and permissions
* Explore PowerShell automation
* Develop professional documentation habits

## Lab Environment

### Host Machine

* Windows 11
* Intel Core i5
* 16 GB RAM

### Virtualization Platform

* VirtualBox

### Planned Infrastructure

* DC1 (Domain Controller)
* FILE1 (File Server)
* HELPDESK1 (Ticketing System)
* UBUNTU1 (Linux Server)
* CLIENT1 (Windows 11 Client)
